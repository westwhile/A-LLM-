# 实现状态与改进清单

## 数据工程

已实现：
- 本地合成样例数据生成与读取。
- schema 校验、日期规范化、复权价格和收益率计算。
- 基础股票池过滤接口。

未实现：
- 真实 AkShare 数据拉取和字段映射。
- 历史指数成分、ST、停复牌、涨跌停、退市数据的 point-in-time 数据表。
- Parquet 存储和数据版本记录。

可改进：
- 增加数据质量报告，检查主键唯一性、缺失率、异常价格和成交额。
- 引入交易日历缓存，避免依赖单表推断交易日。

## 因子构建与处理

已实现：
- 量价、估值、规模、质量、成长、资金流、LLM 事件因子示例。
- MAD 去极值、截面 z-score、行业/市值中性化。
- 因子相关性计算。

未实现：
- IDIO_VOL、更多资金流明细和完整基本面因子。
- 财务数据真实公告日、修订数据和可用日校验。

可改进：
- 添加覆盖率、行业覆盖、市值分桶覆盖和缺失模式分析。
- 对因子方向和入模筛选做 walk-forward 约束。

## 因子检验

已实现：
- IC、Rank IC、ICIR、命中率、t 值。
- 分组收益和多空组差。

未实现：
- 分年度 IC、滚动 IC、因子衰减图。
- 牛熊震荡市场环境分段。

可改进：
- 增加样本内/样本外切分和参数敏感性报告。
- 增加 turnover 与容量约束分析。

## 组合与回测

已实现：
- TopN 等权组合。
- 下一交易日生效的日频净值回测。
- 佣金、印花税、滑点、冲击成本的基础成本扣减。
- 年化收益、波动率、夏普、Calmar、最大回撤、换手率。

未实现：
- 真实下一交易日开盘价成交。
- 涨跌停禁买禁卖后的部分成交和持仓延续。
- 行业偏离、单次调仓最大换手率和现金管理。

可改进：
- 使用订单/成交明细表记录每次调仓。
- 增加不同成本情景和延迟成交压力测试。

## 风险归因

已实现：
- 行业暴露聚合接口。
- 绩效指标 CSV 输出。

未实现：
- 行业归因、市值暴露、回撤区间归因。
- 基准对齐后的 Alpha、Beta、跟踪误差和信息比率完整报告。

可改进：
- 加入年度表现、月度收益热力图和行业暴露图。

## LLM 辅助模块

已实现：
- 事件标签 JSON 字段校验。
- 情绪得分和置信度加权聚合。
- 明确不将 LLM 标签作为核心买卖信号。

未实现：
- LLM API 调用、prompt 模板和人工抽查流程。
- 公告/新闻文本抓取。

可改进：
- 增加 50 条人工抽查样本和一致性评估。
- 对典型事件生成报告案例，而不是自动形成投资结论。

## 报告与展示

已实现：
- README、配置文件、核心代码、测试和实现状态文档。
- 样例流水线会输出指标、IC、分组收益、相关性和净值 CSV。

未实现：
- 完整 3000 字研究报告、PDF 导出和图表。
- Notebook 顺序复现全流程。

可改进：
- 在真实数据接入后补齐图表和结论，报告中单独说明生存者偏差、交易不可达和成本压力测试。
