# A-LLM-
A股多因子选股策略研究｜Python、Pandas、Statsmodels、AkShare/Tushare 基于A股市场构建多因子选股研究框架，采集并清洗行情、估值、财务、行业、停牌、ST和涨跌停等数据，构建动量、反转、波动率、流动性、估值、质量、成长和资金流等因子。对因子进行去极值、标准化、行业与市值中性化处理，并通过IC、Rank IC、ICIR、分组回测和因子衰减检验评估因子有效性。进一步构建月度调仓Top50多因子组合，加入交易成本、滑点和交易约束，完成与基准指数的收益、回撤、夏普比率、信息比率和换手率对比。项目同时尝试使用LLM对公告和新闻进行事件标签提取，用于辅助解释策略阶段性表现。
